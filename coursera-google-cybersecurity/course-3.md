# Course 3 – Connect and Protect: Networks and Network Security

Google Cybersecurity Professional Certificate

---

## Overview

This course focused on the fundamentals of computer networking and the security measures used to protect modern network infrastructures. It introduced how devices communicate, how data travels across networks, and how cybersecurity professionals secure network environments against threats and unauthorized access.

---

## Core Concepts Covered

* Networking Fundamentals
* Network Operations
* Network Devices & Structure
* OSI and TCP/IP Models
* Network Protocols
* IP and MAC Addresses
* Firewalls
* Virtual Private Networks (VPNs)
* Proxy Servers
* Intrusion Detection Systems (IDS) & Intrusion Prevention Systems (IPS)
* Network Security Hardening
* Network Intrusion Tactics
* Network Attack Surface and Vulnerabilities

## Networking Fundamentals

Learned the basic principles of computer networking and how interconnected devices communicate to exchange information.

Topics included:

* Types of networks (LAN, WAN, WLAN, MAN)
* Network topology concepts
* Client-server architecture
* Data transmission fundamentals
* Network communication processes

## Network Operations

Studied how data travels across networks and the processes required for successful communication.

Key concepts:

* Packet transmission
* Data encapsulation and decapsulation
* Network traffic flow
* Communication pathways
* Reliability and efficiency in network communications

## Network Devices & Structure

* Routers
* Switches
* Hubs
* Modems
* Access Points
* Servers

Learned how these devices work together to create and maintain network connectivity.

## OSI & TCP/IP Models

Explored the foundational networking models used to understand network communication.

### OSI Model

* Physical Layer
* Data Link Layer
* Network Layer
* Transport Layer
* Session Layer
* Presentation Layer
* Application Layer

### TCP/IP Model

* Network Access Layer
* Internet Layer
* Transport Layer
* Application Layer

These models provide a framework for understanding data movement and security responsibilities across different layers.

## Network Security Controls

### Firewalls

Studied how firewalls protect networks by controlling incoming and outgoing traffic.

Topics included:

* Packet filtering
* Traffic inspection
* Access control rules
* Security boundaries

### Virtual Private Networks (VPNs)

Learned how VPNs provide secure communication channels through encryption and secure tunneling.

### Proxy Servers

Explored how proxy servers:

* Forward network requests
* Improve privacy
* Filter traffic
* Support organizational security policies

### IDS & IPS

Introduced to:

* Intrusion Detection Systems (IDS)
* Intrusion Prevention Systems (IPS)

Learned how these technologies help detect and respond to malicious network activity.

* Reconnaissance
* Packet sniffing
* Spoofing attacks

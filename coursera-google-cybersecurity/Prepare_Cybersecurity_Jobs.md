# Course 8 – Put It to Work: Prepare for Cybersecurity Jobs

Google Cybersecurity Professional Certificate

---

## Overview

This capstone course focused on applying the knowledge and skills gained throughout the Google Cybersecurity Professional Certificate. It involved completing hands-on security tasks, documenting investigations, communicating findings, and preparing for entry-level cybersecurity roles.

---

## Core Concepts Covered

* Security Incident Investigation
* Risk Assessment
* Threat Analysis
* Vulnerability Assessment
* Security Documentation
* Incident Handler's Journal
* Log Analysis
* Security Tools
* Professional Communication
* Cybersecurity Portfolio Development
* Resume Preparation
* Interview Preparation

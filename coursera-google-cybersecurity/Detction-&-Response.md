# Course 6 – Sound the Alarm: Detection and Response

Google Cybersecurity Professional Certificate

---

## Overview

This course focused on how cybersecurity professionals detect, investigate, and respond to security incidents. It introduced security monitoring, log analysis, incident response, SIEM tools, and techniques used to identify, contain, and recover from cyber threats.

---

## Core Concepts Covered

* Security Monitoring
* Security Incident Detection
* Incident Response Lifecycle
* Security Information and Event Management (SIEM)
* Log Analysis
* Security Alerts
* Indicators of Compromise (IOCs)
* Packet Capture (PCAP)
* Network Traffic Analysis
* Digital Forensics
* Incident Documentation
* Containment, Eradication & Recovery

---

## Security Monitoring

Learned how organizations continuously monitor systems, endpoints, and networks to detect suspicious activity and respond to potential security threats.

Topics included:

* Continuous monitoring
* Security events
* Event correlation
* Alert generation
* Threat detection

---

## Security Incident Detection

Developed an understanding of how analysts detect malicious activity using logs, alerts, indicators of compromise, and abnormal system behavior.

Topics included:

* Indicators of Compromise (IOCs)
* Indicators of Attack (IOAs)
* False positives
* Threat identification
* Alert validation

---

## SIEM & Log Analysis

Learned how Security Information and Event Management (SIEM) platforms collect, correlate, and analyze logs from multiple sources to support security investigations.

Topics included:

* SIEM fundamentals
* Log collection
* Event correlation
* Security dashboards
* Log analysis

---

## Network Traffic Analysis

Studied how network traffic is analyzed to identify suspicious communication and potential attacks.

Topics included:

* Packet Capture (PCAP)
* Network protocols
* Traffic inspection
* Suspicious connections
* Network anomalies

---

## Cybersecurity Application Tools

Gained hands-on experience with industry-standard cybersecurity tools used for monitoring, network analysis, packet inspection, web application security testing, and threat detection.

Tools learned:

* Splunk
* Google SecOps
* Wireshark
* Burp Suite
* tcpdump
* Suricata

---

## Incident Response

Explored the complete incident response lifecycle followed by security teams.

Topics included:

* Preparation
* Detection & Analysis
* Containment
* Eradication
* Recovery
* Post-incident review
* Incident Handler's Journal

Learned to create and maintain an **Incident Handler's Journal** to document evidence, investigation steps, observations, and response actions throughout a security incident.
